Hello! For some fun and practice this summer, I've made some games! Would you like to play?

Play CodeNames
This is a Peer-to-Peer implementation of the party game CodeNames. One computer acts as the information available to the codemakers, who make clues for the other players. The other computer contains the information available for the codebreakers, who use the clues given by the codemakers to guess the secret words. If you haven't played CodeNames before, there's a available with all the rules. I wrote this game to be played with two machines in the same room, like the original party game.

Play Logic Puzzles
This game randomly generates an Einstein-style logic puzzle and provides an interactive solving grid.

Calvin's Game
This game was invented by my 9-year-old son. It started with a child's drawing, and has grown into the game here.
